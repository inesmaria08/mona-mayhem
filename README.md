## 🎯 What You'll Build

Transform a simple Astro template into a **fully functional GitHub contribution battle arena** featuring:

- 🏆 **Head-to-Head Comparison**: Pit any two GitHub users against each other
- 📊 **Visual Contribution Graphs**: Beautiful, interactive contribution calendars
- 🎨 **Retro Arcade Theme**: Pixel-perfect 8-bit aesthetics with Press Start 2P font
- ⚡ **Lightning Fast**: Server-side rendered with Astro for optimal performance
- 🎮 **Game-like Experience**: Winner declarations, stats, and battle animations

## 🚀 Quick Start

<div align="center">

### ⚡ Get Coding in 3 Steps

</div>

1. **Fork & Clone** 🍴
   ```bash
   # Click "Use this template" or fork this repo
   git clone https://github.com/YOUR_USERNAME/mona-mayhem.git
   cd mona-mayhem
   ```

2. **Install & Run** 🛠️
   ```bash
   npm install
   npm run dev
   ```

3. **Choose Your Path** 🛤️
   - **VS Code**: Open in editor, start chatting with Copilot
   - **CLI**: Run `copilot` commands in terminal

## 🛠️ Tech Stack & Requirements

### Core Technologies
- **🎯 Framework**: [Astro](https://astro.build/) v5 — The web framework for content-driven websites
- **⚙️ Runtime**: Node.js with [@astrojs/node](https://docs.astro.build/en/guides/integrations-guide/node/) adapter
- **🎨 Font**: Press Start 2P (Google Fonts) — Authentic retro gaming typography
- **📡 API**: GitHub's contribution graph API — Real contribution data

### Prerequisites
- ✅ GitHub Copilot (Pro, Business, or Enterprise)
- ✅ Git & GitHub account
- ✅ Node.js (LTS recommended)

### Track-Specific Requirements

#### 🖥️ VS Code Track
- VS Code v1.107+
- GitHub Copilot extension (signed in)
- Chat panel ready

#### 💻 CLI Track
- GitHub Copilot CLI (`copilot`)
- Node.js 22+ (for npm install) or Homebrew/WinGet
